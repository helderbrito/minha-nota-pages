Política de Privacidade

Nome do Aplicativo: Minha Nota
Desenvolvedor: Equipe Minha Nota

Esta Política de Privacidade descreve como o aplicativo Minha Nota coleta, usa e protege suas informações. Ao utilizar este aplicativo, você concorda com as práticas descritas nesta política.

1. Uso da Câmera

O aplicativo Minha Nota utiliza a câmera do seu dispositivo para escanear códigos QR presentes em notas fiscais eletrônicas (NFCe). A câmera é necessária para que você possa digitalizar as notas fiscais e processá-las dentro do aplicativo.

2. Processamento de Imagens do QR Code

É importante destacar que:

  • Nenhuma imagem do QR Code é enviada para servidores externos. Todo o processamento das imagens capturadas pela câmera para leitura de QR Code é realizado exclusivamente no seu dispositivo.
  • As imagens capturadas são utilizadas apenas para extrair as informações do código QR e não são armazenadas permanentemente no dispositivo.

3. Cadastro e Conta de Usuário

Para utilizar as funcionalidades completas do aplicativo, você pode optar por criar uma conta. Durante o cadastro, coletamos as seguintes informações:

  • Nome e sobrenome
  • Apelido
  • Endereço de e-mail
  • CPF
  • Telefone
  • Cidade e Estado
  • Foto de perfil (opcional)

Esses dados são transmitidos de forma segura para nossos servidores e são utilizados exclusivamente para identificação do usuário, autenticação e prestação de suporte.

4. Armazenamento de Dados

Os dados do aplicativo são armazenados de duas formas:

  • Dados locais: As informações das notas fiscais escaneadas são armazenadas localmente no seu dispositivo, em um banco de dados seguro.
  • Dados na nuvem: Os dados de cadastro (perfil do usuário) e foto de perfil são armazenados em nossos servidores seguros para permitir autenticação e sincronização.

Você tem controle total sobre seus dados e pode excluí-los a qualquer momento através das funcionalidades do aplicativo.

5. Consulta de Notas Fiscais

Ao escanear um QR Code de NFCe, o aplicativo consulta os portais oficiais do governo para obter os dados detalhados da nota fiscal. Esta consulta é realizada diretamente do seu dispositivo, sem intermediação de nossos servidores.

6. Publicidade

O aplicativo Minha Nota utiliza o serviço Google AdMob para exibição de anúncios. O Google AdMob pode coletar informações do dispositivo, como identificador de publicidade, para personalizar os anúncios exibidos. Para mais informações sobre como o Google utiliza dados, consulte a Política de Privacidade do Google.

7. Permissões Necessárias

O aplicativo solicita as seguintes permissões:

  • Câmera: Necessária para escanear códigos QR das notas fiscais.
  • Armazenamento: Para salvar fotos de perfil e dados locais do aplicativo.
  • Internet: Para consultar notas fiscais, autenticação e exibição de anúncios.

8. Segurança

Implementamos medidas de segurança para proteger suas informações:

  • Suas credenciais são armazenadas de forma segura no dispositivo.
  • Toda comunicação é realizada por conexão criptografada.
  • Seguimos as melhores práticas de segurança da indústria.

9. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de marketing. Os dados podem ser compartilhados apenas:

  • Com provedores de serviço que auxiliam na operação do aplicativo (hospedagem, infraestrutura).
  • Quando exigido por lei ou ordem judicial.

10. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que você revise esta página ocasionalmente para se manter informado sobre como protegemos suas informações.
